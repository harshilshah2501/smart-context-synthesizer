Context Synthesizer — install card
==================================

Ubuntu 22.04 / WSL2 — pick one way to get the toolkit:

  A) Git clone
     git clone https://github.com/harshilshah2501/smart-context-synthesizer.git
     cd smart-context-synthesizer/context-synthesizer
     bash install.sh your.handle

  B) Release tarball
     tar -xzf context-synthesizer-toolkit-*.tar.gz
     cd context-synthesizer-toolkit-*

Then:
  bash run-setup.sh your.handle

Verify:
  csynth doctor
  csynth dashboard

Toggle proxy (no reinstall):
  csynth proxy on | csynth proxy off

Claude Max/Pro login only — no API key.

Docs: docs/guides/DEVELOPER_ONBOARDING.md

Build tarball:
  bash packaging/build-release-tarball.sh
